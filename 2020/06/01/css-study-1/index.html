黑马前端 学成在线案例（盒子+浮动）

总体心得

  1. 首先，事先根据标准流在纵向分成大致几个部分，如案例中 的header banner goods box footer五个部分；然后对每个行细分列，这些列的排布都是浮动实现的；要仿制网站的话，首先要看情况切图，在visio上画好每个块。
  2. css有一些通用的代码，如去除内外边距、清除浮动、去除链接下划线，可以事先写出来。实际编写时，先写好该部分的html代码，然后写最外层盒子的样式，写好后对其中的每个列块，直接先确定浮动方式、宽高，注意宽高除了内容需要动态变化的移动需要指定。这些确定后，大致布局确定，然后再具体写每个列小盒子内容的样式。
  3. 对我而言，一个难点是确定margin，并不能根据设计图准确地确定margin；还有就是当多个类型标签嵌套时，不清楚该给谁样式，如li套a，那么样式该给li还是a，不好确定。

局部问题

  1. nav和subnav样式设置的区别
  2. course中多行文本（不同标签）的显示
  3. box-bd细节设置，为保证内容动态添加不设高度，因此必须清楚浮动
  4. 父右子左 links

一些通用写法

  1. 通用部分
  2. 盒子阴影
  3. 透明背景
