2020.3.5 以太坊环境搭建

一、windows 以太坊环境搭建

  1. 参考：
    https://blog.csdn.net/salmonwilliam/article/details/84678261
    https://github.com/ethereum/go-ethereum/wiki/Installation-instructions-for-Windows
  2. geth客户端 官网无法下载最新版1.9.11 只能安1.8.3
    环境变量变化
    系统变量Path新增 D:\Work\Ethereum\Geth
  3. 安好了go环境
  4. geth version 显示
    Geth
    Version: 1.8.3-stable
    Git Commit: 329ac18ef617d0238f71637bffe78f028b0f13f7
    Architecture: amd64
    Protocol Versions: [63 62]
    Network Id: 1
    Go Version: go1.10
    Operating System: windows
    GOPATH=D:\Project\go\go_work
    GOROOT=C:\go
  5. geth 启动默认连接主网
  6. geth –datadir .\data
  7. 启动节点同步
    geth –-datadir . –syncmode fast
    fast是同步数据 会下区块 不验证
    geth –testnet –datadir . –syncmode fast
  8. 一个网络对应一条链 一个ID

二、windows go环境搭建

  1. 参考：
    下载链接：https://golang.google.cn
    下载方式：手机迅雷
  2. 环境变量的变化参考：
    https://www.jianshu.com/p/602c9ed4847d
    用户变量
    多了 GOPATH %USERPROFILE%\go
    PATH %USERPROFILE%\go\bin 此处表示go代码的工作路径
    系统变量
    多了 PATH D:\Work\Go\bin
    根据 echo %USERPROFILE% 知 %USERPROFILE% 为 C:\Users\Administrator>
    修改 用户变量 GOPATH 为 D:\Project\go\go_work
    修改 用户变量 GOPATH 为 D:\Project\go\go_work
